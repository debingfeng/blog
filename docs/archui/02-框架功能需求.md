# 2、Archui框架第一版功能需求

[[toc]]

> 基于Vue2.x版本，以雅虎Pure.css为基础扩展的ui框架。我们设计丰富主题色彩，设计字体，丰富了复杂响应式网站所需要的UI组件；我们增加更多的通用性模板，针对各行各业以及不同的产品特点做了大量工作。


- **slogan**

> 轻量,支持按需导入、样式名称语义化（可高度自定义）、易扩展、响应式、移动优先适用于任何Web项目。


- 使用场景，丰富的样例模板


    - 电子商城
    
    - 邮箱
    
    - 资讯博客
    
    - 企业响应式产品官方网站
    
    - 移动宣传H5网站
